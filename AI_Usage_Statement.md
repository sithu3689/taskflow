# AI Usage Statement

## AI Tool USED
The following AI tools were used during the development of this project.


##1.ChatGPT

ChatGPT was used as a development assistant for:

- Explaining technical concepts and programming topics.
- Assisting with debugging frontend and backend issues.
- Providing guidance for implementing specific React, Next.js, Express.js and Prisma features.
- Assisting with UI improvements and frontend layout suggestions.


##2.Figma AI

Figma AI was used for:

- Generating user interface design concepts.
- Creating dashboard layout ideas.
- Providing visual inspiration for the frontend interface.
